# Dr. Gowd's Dental Hospital - Landing Page

## Requirements

- **Node.js**: Version **22** is required to run and build this project.

## Getting Started

Follow the steps below to setup, run, and build the project:

### 1. Install Dependencies
After ensuring you are on Node v22, install the project dependencies:
```bash
npm install
```

### 2. Run Locally
To spin up the local development server, run the following command:
```bash
npm run dev
```

### 3. Build for Production
To create a production-ready build, run:
```bash
npm run build
```
